# Промпт для Qoder IDE — верстка PANDOROOM

## Задача
Сверстать 4 страницы сайта квеструма "PANDOROOM" (Владивосток) по макетам из папки `design/`.
Верстка должна быть максимально приближена к макету (pixel-perfect).

## Макеты (1 картинка = 1 страница)
Файлы лежат в `design/`:
1. `Главная.jpg` — главная страница
2. `КАРТОЧКА квеста.jpg` — страница квеста (пример: "Чумной Доктор")
3. `Квесты ОБЩАЯ.jpg` — каталог всех квестов
4. `ПРАЗДНИК ГИД.jpg` — страница детских праздников

## Технологии
- Чистый HTML5 + CSS3 + JS (vanilla)
- CSS-переменные из `styles/variables.css`
- Mobile-first адаптив
- БЭМ-нейминг

## Шрифты
- **Заголовки:** Actay Wide (локальный шрифт, подключить через `@font-face` из `assets/fonts/`)
- **Обычный текст:** Manrope (Google Fonts)
- Подключить Manrope: веса 400, 500, 600, 700

## Цветовая схема (уже в variables.css)
- Фон страницы: `#0a0a0a`
- Фон карточек: `#141414`
- Акцентный зеленый (лайм): `#b5e61d`
- Акцентный розовый/пурпурный: `#d946ef`
- Белый текст: `#ffffff`
- Приглушенный текст: `#9ca3af`
- Границы: `#2a2a2a`

## КРИТИЧЕСКИ ВАЖНО: Эффект заголовков
Часть заголовков имеет эффект многослойной обводки (см. скриншот эффекта):
- Белый текст
- Зеленая обводка/тень смещена влево-вверх
- Розовая обводка/тень смещена вправо-вниз

Реализовать через CSS `text-shadow` с несколькими слоями:
```css
.title-effect {
  color: #ffffff;
  text-shadow:
    -3px -3px 0 #b5e61d,
    -2px -2px 0 #b5e61d,
    3px 3px 0 #d946ef,
    2px 2px 0 #d946ef;
}
```
Если text-shadow недостаточно точно — использовать псевдоэлементы `::before`/`::after` с абсолютным позиционированием и z-index.

## Общие блоки (повторяются на всех страницах)
- **Header:** логотип PANDOROOM (слева), меню (Квесты, Праздники, Кофе, Игровая для детей), телефон/соцсети (справа)
- **Footer:** ссылки на разделы, контакты, соцсети, копирайт
- **Карточки квестов:** изображение, название, цена, сложность, возраст, кнопка
- **Блок "Новости и акции":** 4 карточки с картинками
- **Блок "Отзывы гостей":** карточки с именем, датой, звездами, текстом

## Что верстать на каждой странице

### 1. Главная (index.html)
- Hero-секция с большим заголовком "Самый большой квеструм и площадки для праздников во Владивостоке" + кнопки

- Новости и акции (4 карточки)
- Отзывы гостей (карусель/слайдер)
- Footer

### 2. Каталог квестов (quests.html)
- Заголовок "Выберите квест во Владивостоке"

- Сетка карточек квестов (по 4 в ряд на десктопе)
- Каждая карточка: изображение, название, цена, время, сложность, возраст

- Отзывы гостей
- Footer

### 3. Страница квеста (quest-card.html)
- Название квеста + хлебные крошки
- Галерея изображений (основное фото + миниатюры)
- Описание квеста (табы: Описание, Безопасность, Дополнительные услуги)
- Инфо-блок: сложность, с актерами/без, время, возраст, адрес, цена
- Блок расписания (таблица с датами и временными слотами)
- Отзывы гостей
- Новости и акции
- Footer

### 4. Праздники (holidays.html)
- Hero с заголовком "Площадки для праздников и самый большой квеструм"
- Секция детских праздников по возрастам (малыши, 6-10 лет, 10-15 лет)
- Доп. поводы (выпускные, поступление в школу)
- Иконки услуг
- Квесты с актерами / без актеров / для детей (как на главной)
- Новости и акции
- Отзывы гостей
- Footer

## Правила работы с медиа
- **НЕ** искать картинки в интернете
- **НЕ** использовать placeholder-сервисы
- Для карточек квестов и новостей оставить правильные `<img>` с путями типа `assets/images/quests/quest-name.jpg`
- После верстки составить список нужных медиа-файлов и запросить у пользователя
- SVG-иконки можно сделать inline (простые) или оставить плейсхолдеры с комментариями

## Адаптив
- Mobile-first
- Брейкпоинты: 640px, 768px, 1024px, 1280px
- На мобильных: карточки в 1-2 колонки, меню в бургер, фильтры сворачиваются

## Важно
- Темная тема — везде темные фоны, светлый текст
- Карточки с закругленными углами (~16px)
- Кнопки с закруглением и яркими цветами (зеленый/розовый)
- Hover-эффекты на интерактивных элементах
- Плавные переходы (transition)
